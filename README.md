## What is this?

MEDCOMPANION is a simple web app to help people manage medicines better.

It lets patients track their doses, family members monitor them, and doctors get a quick overview — all in one place.

---

## Features

- 💊 Add and track medicines  
- ⏰ Automatic dose reminders  
- 📊 Basic adherence tracking  
- 👨‍👩‍👧 Family monitoring  
- 👨‍⚕️ Doctor dashboard  
- 🤖 AI chat for medicine-related questions  
- 🚨 SOS button for emergencies  
- 🌍 Multi-language + voice support  

---

## Run locally

1. Install dependencies
   npm install

2. Add your API key in `.env`
   VITE_GEMINI_API_KEY=your_api_key

3. Start the app
   npm run dev
